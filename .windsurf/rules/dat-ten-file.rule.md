# QUY TẮC ĐẶT TÊN FILE ĐÍNH KÈM

Khi tải file đính kèm từ trang chi tiết văn bản, Cascade PHẢI tuân thủ:

## Công thức chung:
{so_hieu_chinh} - {mo_ta} - {ten_file_goc}

## Chi tiết theo loại file:

| Loại file | Mô tả (mo_ta) | Ghi chú |
|-----------|--------------|---------|
| PDF ký số (có text) | {trich_yeu_ngan} - {ngay_ky} | Đọc nội dung để lấy trích yếu |
| Word (.docx) | {trich_yeu_ngan} - ban word | Dùng trích yếu của văn bản chính |
| Excel (.xls,.xlsx) | {trich_yeu_ngan} - bang tinh | Dùng trích yếu của văn bản chính |
| Hình ảnh (.jpg,.png) | {trich_yeu_ngan} - hinh anh | Dùng trích yếu của văn bản chính |
| File nén (.rar,.zip) | file nen - {ten_file_goc} | Giữ nguyên tên gốc |
| PDF scan (ko đọc text) | phu luc - {ten_file_goc} | Giữ nguyên tên gốc |
| File khác | dinh kem - {ten_file_goc} | Giữ nguyên tên gốc |

## Ví dụ với văn bản 1109/SGDĐT-VP:
- PDF ký số: 1109-SGDĐT-VP - trien khai hoc bong Thap Sang Niem Tin - 08042026.pdf
- Word: 1109-SGDĐT-VP - trien khai hoc bong - ban word.docx
- PDF scan: 1109-SGDĐT-VP - phu luc - 7811_signed.pdf
- Excel: 1109-SGDĐT-VP - trien khai hoc bong - bang tinh.xlsx
